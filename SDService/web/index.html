SDService

This service takes sets of words stored into multiple profiles of a Personal Service (PS) record and looks for documents having those words (similar documents) in a database, for example, LILACS+MEDLINE.

Services:

Add profile

Add PS service profile words

Syntax: SDService/?psId=<id>&addProfile=<id>&sentence=<sentence>

psId: personal service record identifier
id: profile identifier
sentence: a sentence with the words stored in the profile

Example: SDService/?psId=Joao Doria&addProfile=books&my preferences are: sentence=romance, adventure, scifi

Delete profile

Delete in this service a PS record profile

Syntax: SDService/?psId=<id>&deleteProfile=<id>

psId: personal service record identifier
id: profile identifier

Example: SDService/?psId=Joao Doria&deleteProfile=books

Get similar documents

Retrieve documents having most of words stored in a set of profiles

Syntax: SDService/?psId=<id>&getSimDocs=<profile>,..,<profile>&outFields=<field>,...,<field>

psId: personal service record identifier
profile: profile identifier
field: tag of similar doc field to be outputed. If no fields were specified all fields will be outputed

Example: SDService/?psId=Joao Doria&getSimDocs=books,cars,hobbies&outFields=id,date,name

Show profiles

Show all profiles and respective words of a PS record

Syntax: SDService/?psId=<psId>&showProfiles=<id>

psId: personal service record identifier
id: profile identifier

Example: SDService/?psId=Joao Doria&showProfiles=
